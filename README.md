FAQ Accordion
A simple FAQ section built with HTML and CSS, replicating a Figma design. One item is shown in its expanded state, with the question highlighted and the answer visible underneath.
Files

index.html - markup for the FAQ card and its five questions
style.css - styling, layout, and the expanded-state treatment

Structure
The page is a centered white card on a purple gradient background. Inside the card:

A bordered "FAQ" heading
A list of question rows, each with a question label and a chevron icon
One row (What is the maximum file upload size?) is marked as active, showing a blue outline around the question and an answer paragraph below itreadme template
